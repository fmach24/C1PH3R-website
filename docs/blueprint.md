# **App Name**: C1PH3R Hub

## Core Features:

- Landing Page: Showcase information about the Cipher student research group.
- Team members: Display information about team members with their photo, name, and role.
- Research Areas: Display research interests: network security, data and application security, threat analysis, cryptography, and penetration testing.
- External Links: Link to the Cipher group's profiles on LinkedIn and Facebook.
- Contact Information: Provide contact information for the faculty supervisor, dr hab. inż. Marcin Niemiec, prof. AGH.

## Style Guidelines:

- Primary color: Deep purple (#483D8B), drawing inspiration from the fingerprint gradient in the group's logo, signifying cybersecurity and digital identity. 
- Background color: Light gray (#F0F0F0) to ensure content readability and provide a clean, modern feel.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights, complementing the primary purple with a contrasting, vibrant touch.
- Headline font: 'Orbitron', sans-serif, as specified by the user for the name of the group: "Koło naukowe C1PH3R". Note: currently only Google Fonts are supported.
- Body font: 'Inter', sans-serif, to provide a clean, readable style for the main content of the site.
- Use security-themed icons to visually represent the group's areas of expertise such as padlocks, shields, or data encryption symbols.
- A modern, single-page layout to quickly present key information about the Cipher research group.
